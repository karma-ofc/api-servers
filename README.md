# Контрольная работа FastAPI

Выполнены все 7 заданий контрольной работы.

## Запущенное приложение

Для запуска используйте:
```bash
uvicorn main:app --reload
```

Сервер будет доступен по адресу: http://127.0.0.1:8000

## Выполненные задания

| № | Задание | Маршрут | Метод |
|---|---------|---------|-------|
| 1.1 | Приветствие | `/info` | GET |
| 1.2 | HTML страница | `/` | GET |
| 1.3* | Калькулятор | `/calculate` | POST |
| 1.4 | Пользователь | `/users` | GET |
| 1.5* | Проверка возраста | `/user` | POST |
| 2.1 | Отзывы (базовая) | `/feedback` | POST |
| 2.2* | Валидация отзывов | `/feedback` | POST |
